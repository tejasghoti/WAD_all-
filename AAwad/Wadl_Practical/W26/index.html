PICT B.E. Course Structure

FE (First Year)

Engineering Mathematics I/II: Calculus, linear algebra, differential equations.

Engineering Physics: Mechanics, optics, modern physics.

Engineering Chemistry: Material science, electrochemistry.

Programming & Problem Solving: C programming and basic algorithms.

Engineering Mechanics: Statics and dynamics of rigid bodies.

SE (Second Year)

Data Structures & Algorithms: Arrays, linked lists, trees, graphs, sorting.

Discrete Mathematics: Logic, sets, relations, graphs.

Database Management Systems: Relational model, SQL, normalization.

Object-Oriented Programming: Java/C++ OOP concepts.

Computer Networks & Security: OSI/TCP-IP model, routing, network security basics.

TE (Third Year)

Operating Systems: Processes, scheduling, memory management, file systems.

Theory of Computation: Automata, grammars, Turing machines.

Software Engineering: SDLC, agile, design patterns, testing.

Artificial Intelligence: Search, knowledge representation, ML basics.

Web App Development & Cloud Computing: Full-stack web dev and cloud platforms.

BE (Final Year)

Data Science & Big Data Analytics: Data pipelines, Hadoop, Spark.

High Performance Computing: Parallel computing, GPU programming.

Deep Learning: Neural networks, CNNs, RNNs, transformers.

Final Year Project: Capstone software project under faculty mentorship.

Electives: Blockchain, IoT, NLP, Information Security.
