PICT B.E. Curriculum

Computer Engineering - Course Structure

FE - First Year

  • Engineering Mathematics

    Calculus, linear algebra, and differential equations.

  • Basic Electrical Engineering

    Fundamentals of circuits, AC/DC systems, and transformers.

  • Programming & Problem Solving

    Introduction to C programming, logic building, and algorithms.

SE - Second Year

  • Data Structures

    Arrays, linked lists, trees, graphs, and sorting algorithms.

  • Object Oriented Programming

    OOP concepts using Java: inheritance, polymorphism, abstraction.

  • Computer Graphics

    2D/3D transformations, clipping, and rendering techniques.

TE - Third Year

  • Database Management Systems

    SQL, normalization, ER diagrams, and transaction management.

  • Web Application Development

    HTML, CSS, JavaScript, Node.js, Express, Angular, and MongoDB.

  • Computer Networks

    OSI model, TCP/IP, routing protocols, and network security.

BE - Final Year

  • Machine Learning

    Supervised/unsupervised learning, neural networks, and model evaluation.

  • Cloud Computing

    AWS, virtualization, containerization, and cloud service models.

  • B.E. Project

    Capstone project applying knowledge from all four years to solve a real-world problem.
